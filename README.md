# すごろくアプリ

Windows Formアプリケーションで作成したすごろくアプリのソースコードを管理するリポジトリです。

## 開発環境

**OS**  
Windows10 Pro

**IDE**  
Visual Studio 2017

**プログラム言語**  
C#

**フレームワーク**  
.NET Freamwork 4.6.1

## プレイ方法

[Sugoroku.exe](https://github.com/hiurawork/Sugoroku/releases/download/1.0.0/Sugoroku.exe)をダウンロードして、実行してください。

※ 動作確認環境：Windows 10 Pro
